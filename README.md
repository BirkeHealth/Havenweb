# Haven Health Insurance Marketing Site

## Overview
Production-ready static marketing site scaffold for Haven Health Insurance (`havenhealthinsurance.com`) with a luxury-minimal navy/gold brand system and shared multi-page layout.

## Local Dev
No build step required. Open `/home/runner/work/Havenweb/Havenweb/index.html` directly in a browser.

## Deploy to Render
1. Connect the repository in Render.
2. Keep static publish path as `./`.
3. Render reads `/home/runner/work/Havenweb/Havenweb/render.yaml` for routing and headers.

## Brand Tokens
